Unity - +

Change play mode tint - +

The biggest mistake that I see a lot of people do when first starting out with Unity is hitting play, testing their game, and then making changes to anything. The problem is that they forget to exit play mode which means that their changes won't save. This tip can help you avoid those situations.

1 Go to Edit > Preferences > Colors > Playmode tint

2 Change the colour from grey to anything else, I personally go for a subtle red. If you hit play and then change it, you'll be able to see the tint working.

Game Engines - +

The Game Engine is probably the most important tool when it comes to creating a game. The engine usually dictates what programming language the game's code will be written in and what platforms the game will support.

Unity 2D / 3D - +

Languages C#

Supported Platforms Desktop Console Mobile Web VR More

Unity is the Jack of all trades when it comes to Game Engines as it can be used to create pretty much any type of game for pretty much any platform. Unity is also free to download and use (Plus and Pro versions come with additional support) which makes it great for beginners who are getting started with game making. Since Unity is free, it also means that there's a lot of support that's out there making learning the engine relatively easy.

Godot 2D / 3D

Languages GDScript C# C++ Visual Script More

Supported Platforms Desktop Mobile Web

The open source alternative to Unity, which means that it has a huge community of developers working on it, making this one of the fastest growing engines. You too can modify it in any way you want. There's support for a big range of languages like C++, C#, Visual Scripting and even a custom made language made just for Godot called GDScript. It has support for many platform, and you can even write custom builds yourself. Did I mention that the whole engine only takes up 20MB?

Frameworks - +

Frameworks are essentially stripped down versions of Game Engines. In exchange for fancy features, you get a lower level access which means better performance.

Love 2D

This is lightweight framework you can use to make games using Lua, it doesn't have a GUI which means you need to script everything yourself, and yet it's quite easy to make simple games.

DragonRuby

DragonRuby is another lightweight game toolkit which you can use to make games in Ruby. It's created by Ryan C. Gordon who's behind SDL.

SDL

SDL is the OG framework and is used pretty much everywhere, it was created in 1998 and has been receiving updates since, it's been used to create popular games like Half-Life.

What exactly is the Holy Hand Book?

The Holy Hand Book is a website that contains different resources to help you with game development. You'll find everything from tips and tricks to links to software and other resources.

Why the name "Holy Hand Book"?

I was writing this project just after I had seen ready player one. For some reason, the reference to the holy hand grenade really stuck with me. So when I was coming up with a name for this project, that was the first thing that popped into my head, and that was that.

How did this come about?

This project originally started out when my best friend asked me for tips to help him with Unity. I created a text file with tons of tips, tricks and links and realised that a website would be a much better format for this. So here we are.
