Programs

Game Engines

Unity - A very versatile engine that can be used for both 3D and 2D games. I would highly recommend this for beginners.

Godot - The open source alternative to Unity, can do just as much if not more than Unity and due to it being open source, you can modify it in any way you want. Did I mention that it only takes up 50MB?

Love 2D - This is a 2D game engine.

IDEs

JetBrains - JetBrains for programmers is what Adobe is for artists. They make tools for pretty much every programming language and they're my favourite. All of the programs have a consistent format which means that transitioning between them is really simple. However just like Adobe, you'll need to pay a monthly or yearly fee. But they have a lot of discounts like a free licence for anyone in education.

Visual Studio - Visual Studio is a free IDE that you can use for pretty much anything.

Art Tools

Photoshop - Photoshop is Jack of all art tools. It can be used for any type of art, from traditional to pixel which means it'll work with any project you want.

Aseprite - Aseprite is my favourite pixel art tool by far, it's very versatile and has tons of functionality. While at the same it's specialised for pixel art which means everything is focused on manipulating pixels.

About

This sections contains all the information about Holy Hand Book

What exactly is the Holy Hand Book?

The Holy Hand Book is a website that contains different resources to help you with game development. You'll find everything from tips and tricks to links to software and other resources.

How did this come about?

This project originally started out when a friend asked me for tips to help him with Unity. I created a text file with tons of tips, tricks and links and realised that a website would be a much better format for this.
